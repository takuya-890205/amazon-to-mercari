# Amazon → メルカリ 出品下書き生成ツール

Amazon商品ページのURLを入力するだけで、メルカリ出品用の下書き（タイトル・説明文・カテゴリ・価格・画像）を自動生成するツールです。

## 主な機能

- **ワンクリック取得** — Amazon URLを貼るだけで商品情報を自動取得
- **AI説明文生成** — Gemini AI（無料）が売れやすい説明文を自動作成
- **価格自動計算** — 手数料・送料を差し引いた利益をリアルタイム表示
- **画像ダウンロード** — 商品画像を自動取得・メルカリ最適サイズに変換
- **デフォルトテンプレート** — よく使う設定（配送方法・説明文フッター等）を保存
- **コピー & ダウンロード** — テキストコピー、JSON/テキストでダウンロード

## セットアップ（3ステップ）

### 1. Pythonをインストール

[Python公式サイト](https://www.python.org/downloads/) からダウンロード・インストールしてください。

> **重要**: インストール時に **「Add Python to PATH」にチェック** を入れてください。

### 2. Gemini APIキーを取得（無料）

1. [Google AI Studio](https://aistudio.google.com/apikey) にアクセス
2. Googleアカウントでログイン
3. 「APIキーを作成」をクリック
4. 表示されたキーをコピー

> 無料枠で十分使えます（1分あたり15リクエスト）。

### 3. セットアップを実行

`setup.bat` をダブルクリックしてください。画面の指示に従ってAPIキーを入力すれば完了です。

## 使い方

### 起動

`run.bat` をダブルクリックするとブラウザで自動的に開きます。

### 基本的な流れ

1. サイドバーに **Amazon商品URL** を貼り付け
2. 自動的に商品情報を取得 → AI が下書きを生成
3. タイトル・説明文・価格を必要に応じて編集
4. 「テキストをコピー」でメルカリに貼り付け

### デフォルトテンプレート

サイドバーの「デフォルトテンプレート設定」で以下を保存できます：

- **説明文ヘッダー** — 全商品の説明文冒頭に自動挿入（例: 「【即日発送】【送料無料】」）
- **説明文フッター** — 全商品の説明文末尾に自動挿入（例: 「ご質問はお気軽にコメントください。」）
- **配送方法・サイズ・発送元・発送日数** — いつも使う設定を記憶

### コマンドライン（上級者向け）

```bash
# 基本テンプレートで下書き生成
python main.py "https://www.amazon.co.jp/dp/XXXXXXXXXX"

# Gemini AIで説明文を生成
python main.py "https://www.amazon.co.jp/dp/XXXXXXXXXX" --ai

# 画像もダウンロード
python main.py "https://www.amazon.co.jp/dp/XXXXXXXXXX" --ai --images
```

## APIキーなしでも使えます

Gemini APIキーがなくても基本テンプレートで下書きを生成できます。
AIによる説明文生成をOFFにしてお使いください。

## 注意事項

- 本ツールは個人の出品作業を効率化するためのものです
- Amazonの商品情報は参考情報です。出品前に必ず内容を確認してください
- 大量アクセスはお控えください
