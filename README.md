# Amazon → メルカリ 出品下書き生成ツール

Amazon商品ページのURLを入力するだけで、メルカリ出品用の下書き（タイトル・説明文・カテゴリ・価格・画像）を自動生成するツールです。

## 主な機能

- **ワンクリック取得** — Amazon URLを貼るだけで商品情報を自動取得
- **購入履歴から選択** — Amazon購入履歴を開いて商品を選ぶだけで自動入力
- **AI説明文生成** — Gemini AI（無料）が売れやすい説明文を自動作成
- **価格自動計算** — 手数料・送料を差し引いた利益をリアルタイム表示
- **画像ダウンロード** — 商品画像を自動取得・メルカリ最適サイズに変換
- **メルカリ自動転記** — 出品フォームに自動入力（Playwright使用）
- **デフォルトテンプレート** — よく使う設定（配送方法・説明文フッター等）を保存

## かんたんセットアップ（はじめての方）

### ステップ1: ダウンロード

このページの緑色の **「Code」ボタン → 「Download ZIP」** をクリックして、好きな場所に解凍してください。

### ステップ2: 起動

**`start.bat` をダブルクリック** するだけです。

- Pythonが未インストールなら自動でインストールを案内します
- 初回は必要なソフトを自動でダウンロードします（数分かかります）
- 2回目以降はすぐに起動します

### ステップ3: APIキーを設定（任意・無料）

AIで説明文を自動生成したい場合のみ必要です。

1. [Google AI Studio](https://aistudio.google.com/apikey) にアクセス
2. Googleアカウントでログイン →「APIキーを作成」をクリック
3. アプリのサイドバー「API設定」にキーを貼り付け

> APIキーなしでも基本テンプレートで下書き生成できます。

## 使い方

### 基本的な流れ

1. `start.bat` をダブルクリックして起動
2. **方法A**: サイドバーに Amazon商品URLを貼り付け
3. **方法B**: 「📦 購入履歴から選ぶ」ボタンで購入履歴から商品を選択
4. 自動的に商品情報を取得 → AI が下書きを生成
5. タイトル・説明文・価格を必要に応じて編集
6. 「テキストをコピー」または「メルカリに転記」

### デフォルトテンプレート

サイドバーの「デフォルトテンプレート設定」で以下を保存できます：

- **説明文ヘッダー** — 冒頭に自動挿入（例: 「【即日発送】【送料無料】」）
- **説明文フッター** — 末尾に自動挿入（例: 「ご質問はお気軽にコメントください。」）
- **配送方法・サイズ・発送元・発送日数** — いつも使う設定を記憶

## コマンドライン（上級者向け）

```bash
# 基本テンプレートで下書き生成
python main.py "https://www.amazon.co.jp/dp/XXXXXXXXXX"

# Gemini AIで説明文を生成
python main.py "https://www.amazon.co.jp/dp/XXXXXXXXXX" --ai

# 画像もダウンロード
python main.py "https://www.amazon.co.jp/dp/XXXXXXXXXX" --ai --images
```

## よくある質問

**Q: 「Pythonが見つかりません」と表示される**
A: [Python公式サイト](https://www.python.org/downloads/) からインストールしてください。インストール時に **「Add Python to PATH」にチェック** を入れるのを忘れないでください。

**Q: APIキーは有料ですか？**
A: Gemini APIは無料枠で十分使えます（1分あたり15リクエスト）。

**Q: APIキーなしでも使えますか？**
A: はい。「Gemini AIで説明文を生成」のチェックを外せば、基本テンプレートで下書きを生成できます。

## 注意事項

- 本ツールは個人の出品作業を効率化するためのものです
- Amazonの商品情報は参考情報です。出品前に必ず内容を確認してください
- 大量アクセスはお控えください
